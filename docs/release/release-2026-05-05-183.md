## Publication Status

- Git tag `release-2026-05-05-183` was pushed, but the GitHub Release was not published.
- GitHub Actions run `25411848092` failed in `Verify release inputs` during `Run release verification`.
- Root cause: the Linux runner passed the Rust PATH regression fixed in this candidate, then failed `scripts/release-flow-contract.test.mjs` in the desktop release build runner contract. The failing tests simulated a Windows desktop release plan on an Ubuntu runner but expected `process.execPath` to live beside `node_modules/pnpm/bin/pnpm.cjs`. On GitHub-hosted Ubuntu, Node is installed under `/opt/hostedtoolcache/node/.../bin/node`, and that adjacent pnpm CLI path does not exist, so the actual Windows-simulated plan correctly fell back to `pnpm.cmd` and the test assertion failed before the release matrix could package assets.
- The full release payload from this failed candidate is carried forward into `release-2026-05-05-184`.
